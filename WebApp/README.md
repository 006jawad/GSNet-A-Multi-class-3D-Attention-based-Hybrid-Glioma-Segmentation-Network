## Run Locally

Clone the

```bash
  git clone https://github.com/006jawad/GSNet_.git
```

Go to the project directory

```bash
  cd GSNet_/WebApp
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the Web App

```bash
  flask run
```
